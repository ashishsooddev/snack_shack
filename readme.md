# 🍔 Snack Shack Website

## 📌 Project Overview

Snack Shack is a responsive, multi-page food website designed to showcase a snack restaurant’s menu, services, and brand. The project focuses on clean UI design, smooth navigation, and user-friendly layout using modern web development practices.

---

## 🚀 Features

* 🏠 Home page with hero section and highlights
* 📖 About page describing the brand/story
* 🍟 Menu page displaying food items
* 📱 Fully responsive design for different screen sizes
* 🎨 Styled with custom CSS and reusable components
* 🔗 Social media icons integration
* ⚡ Smooth navigation between pages

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6 Modules)
* Font Awesome (for icons)
